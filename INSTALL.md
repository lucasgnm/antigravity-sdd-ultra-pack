# Quick Start (Copiar/colar no Antigravity)

1) Comece com orquestração SDD:
`@sdd-orchestrator iniciar um novo projeto SDD para web app. Briefing: <cole aqui>.`

2) Se já existe repositório, forneça UMA das opções:
- a) `tree -L 4` (saída)
- b) print das pastas principais
- c) lista de diretórios relevantes

3) Quando o orquestrador pedir, cole as RULES:
- `antigravity/rules/RULES_ENTERPRISE.md`

4) Depois rode a Fatia 0 (scaffold) e valide (npm run dev / equivalente).

5) Siga as iterações: a cada fatia, atualize PRD → Spec → Code Prompts.
