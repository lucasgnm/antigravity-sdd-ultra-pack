# Slice 0 Scaffolder (Web App)

## Goal
Criar a base do projeto (Fatia 0) coerente com a stack: estrutura, navegação, páginas vazias, padrões, qualidade mínima.

## Inputs
- Stack escolhida
- Rotas principais (do PRD)
- Design system (se houver)

## Outputs (mínimo)
- Estrutura de pastas e convenções
- Rotas/páginas com estados (loading/empty/error/success)
- Layout base e navegação
- Setup de lint/typecheck/tests
- Logger básico e error boundary (quando aplicável)
- Como rodar: `npm run dev` e comandos de validação

## Constraints
- Não adicionar features fora do Slice 0.
- Dependências mínimas.
